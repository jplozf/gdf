# gdf - Go Disk Free

A enhanced `df` command written in Go to get free space for all mounted disks, displaying a textual gauge for each.

## Getting Started

1.  **Clone the repository (if applicable):**

    ```bash
    git clone <repository-url>
    cd gdf
    ```

2.  **Run the application:**

    ```bash
    go run main.go
    ```
